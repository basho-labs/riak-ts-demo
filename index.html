Introducing Riak TS

Riak TS is a distributed NoSQL database and key/value store optimized for fast reads and writes of time series data.
It provides a rich set of feature like data co-location and range queries, which make queries faster and easier to run.

    Data Modeling

    Data Modeling

    In order to query time series data, the data is stored in a schema.
    The schema defines what data can be stored in a time series bucket and what type it has.

    Queries can then be written against that schema and the Riak TS query system can validate and execute them.
